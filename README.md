# JobBoard

A SIMPLE JOB BOARD MADE USING SPRING BOOT AND POSTGRESQL.

NOTE:- IF THE PROJECT DOES NOT RUN ON https://jobboard-nbyula.herokuapp.com/user/ , download spring tool suite open it on localhost:8080/user/
